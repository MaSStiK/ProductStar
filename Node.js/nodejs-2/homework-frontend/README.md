### homework-frontend
В этой папке лежит сайт, который взаимодействует с API.

- **index.html** — взаимодействие с двумя API:
  - получение цветовых палитр (Colormagic)
  - поиск по аниме (Jikan)

- **messages.html** — CRUD-эндпоинты:
  - получить список сообщений
  - добавить сообщение
  - изменить сообщение
  - удалить сообщение

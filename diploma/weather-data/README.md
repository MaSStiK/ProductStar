# Прогноз погоды по городу

Небольшой статичный веб‑сайт для просмотра актуальной погоды по названию города.
Проект работает в браузере и использует внешний погодный API для получения данных.

## Возможности

- Поиск погоды по названию города
- Отображение текущей температуры
- Ощущаемой температуры
- Вероятности осадков
- Влажности воздуха
- Скорости ветра
- Текстового описания погодных условий

## Используемые технологии

- HTML5
- CSS3
- JavaScript
- jQuery
- Moment.js
- Weather API

## Запуск как статичный сайт

### Самый простой способ

1. Скачайте или склонируйте проект
2. Откройте файл `index.html` двойным кликом в браузере

⚠️ Некоторые браузеры могут блокировать HTTP-запросы при открытии через `file://`.

### Рекомендуемый способ (через локальный сервер)

#### Вариант 1: через Python

```bash
python -m http.server 8080
```

После этого откройте:

```
http://localhost:8080
```

#### Вариант 2: через Node.js (http-server)

```bash
npx http-server
```

#### Вариант 3: через VS Code

- Установите расширение **Live Server**
- Нажмите **Go Live**

## Примечания

- Сайт является полностью статичным
- Все запросы выполняются из браузера
- API‑ключ виден в клиентском коде — проект предназначен для учебных и демонстрационных целей

# Cards Showcase

Этот проект демонстрирует три различных варианта реализации карточек, выполненные на разных технологиях интерфейсного уровня. Он может служить примерами для учебных задач, внутренней дизайн-системы или в качестве ориентиров при выборе UI-подхода.

## Типы карточек

1. **Styled Components Cards**  
   Карточки реализованы полностью на базе `styled-components`.
   - Динамическое изменение внешнего вида через пропсы (`$active`).
   - Гибкая типографика и кастомизация.

2. **Reactstrap Cards**  
   Карточки построены с использованием библиотеки `reactstrap`.
   - Бутстраповская сетка и готовые UI-компоненты.
   - Минимум кастомного CSS.

3. **Material UI Cards (@mui/icons-material)**  
   Карточки основаны на компонентах Material UI и наборе иконок `@mui/icons-material`.
   - Адаптивность, строгая визуальная система.
   - Богатый набор функциональных UI-элементов.

## Установка

Перейдите в папку с проектом

```bash
npm install
```

## Запуск

```bash
npm run dev
```
